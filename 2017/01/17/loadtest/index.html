loadtest

loadtest 压力测试

可用powershell 或linux bash

命令行如果加post body 会报错

常用参数：

-T 表示持续时间

-c 并行数量

-n 总的请求

-m get,put,post

–rps 每秒发送请求数

-P Request body 数据

-p body 数据存在的文件路径

-H header 添加

-C 设置cookie
