  1. git reflog 命令获取到的内容为本地仓库所有发生过的变更
  2. git branch -m feature/stor-13711 feature/story-13711 修改分支名称
  3. git config –global –edit
  4. git rm –cached 从git索引和缓存中删除
  5. git mv 重命名或者移动文件
  6. git reset –soft “HEAD^” 留下当前变更内容
  7. git reset head 从当前缓存区删除
  8. git checkout . 撤销当前工作区修改
  9. git commit –amend 修改上次提交信息
  10. git log –oneline/–raw
  11. checkout只会移动HEAD指针，reset会改变HEAD的引用值。
  12. git config –global user.name [user.email]
  13. git tag -a ‘annotate’ -m ‘message’
  14. git remote set-url –add 添加多个url
  15. git tag -d
  16. git status –ignored 显示忽略掉文件
  17. git checkout –orphan 新建没有commit的分支
  18. git push origin :refs/tags/ 删除远程tag
  19. git branch -u <origin/mybranch> 设置远程分支
  20. git branch -vv 查看本地分之关联情况
  21. git checkout - 最近两个分子间切换
  22. git commit -m “Big-ass commit” –allow-empty
  23. git commit –amend，撤销上一次提交到暂存区，并重新提交内容；

  24. 修改分支名称
